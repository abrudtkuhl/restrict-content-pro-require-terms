Adds a Terms and Conditions checkbox to the Restrict Content Pro registration form.

== Description ==
Adds a Terms and Conditions checkbox to the Restrict Content Pro registration form. You can even have the label open the terms in a new link if they are not included above or below your form in the content of the page.

== Installation ==
Install and Activate from wordpress.org plugin repository.

It will work with default settings up activation. Check \'Terms\' menu under \'Restrict\' menu section to update default values.
